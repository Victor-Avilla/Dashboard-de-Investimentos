# Dashboard de Investimentos

Projeto pessoal em React + Tailwind CSS que implementa um painel (dashboard) para visualização e acompanhamento de investimentos. Desenvolvido como objeto de estudo pratico com objetivo de consolidar conhecimentos em React e TailwindCSS para um projeto futuro com dados reais e consumo de APIs. A integração de gráficos foi eita com Chart.js e os icones com Lucide.

## Tecnologias
- React
- JavaScript
- Tailwind CSS
- Chart.js
- Vite
- Lucide

## Funcionalidades 
- Página principal (Dashboard) com:
  - Gráficos de linha e pizza para visualização de desempenho e composição do portfólio.
  - Cards de resumo com valores investidos, rentabilidade e alocação por classe de ativo.(dados mockados)
- Componentes reutilizáveis para gráficos e cards.
- Layout responsivo com Tailwind.
